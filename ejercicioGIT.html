Examen Git y Github

Alumn@: Samuel Ortega Perejon

  1. Que es un sistema de control de versiones?

    es una herramienta de software que monitoriza y gestiona cambios en un sistema de archivos.

  2. Cual es la diferencia entre git y github?

    Git es un ejemplo de VCS, y GitHub es un sitio web + infraestructura que proporciona un servidor Git más una serie de herramientas realmente útiles para trabajar con repositorios git individuales o en equipo, como informar problemas con el código, herramientas de revisión, características de administración de proyectos ...

  3. Cómo creo una rama en git?

    cON GIT BRANCH 'NOMBRE RAMA'

  4. Cómo clono un repositorio?

    Con git clone 'url'

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    con git commit

  6. Cómo puedo gener un tag?

    git tag

  7. Cómo hago una mezcla de ramas?

    git merge [nombre-rama]

  8. Cómo puedo moverme a un commit especifico?

    git checkout commitC donde commitC es el hash de dicho commit.

  9. Cómo puedo borrar una rama?

    git branch -d nombre-rama

  10. Cómo puedo borrar un tag?

    Al usar la opción -d y un identificador de etiqueta de git tag

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    git show

  12. Para que sirve el comando git remote -v?

    e utiliza para especificar los extremos remotos sobre los que operarán los comandos de sincronización

  13. Cómo le digo a git cual es mi email y correo?

    Utilizando los siguientes comandos: git config --global user.name "Tu Nombre" git config --global user.email "tuemail@example.com"

  14. Cuales son los pasos para conectar git con github?

    1.Descaragr la version mas reciente de git. 2. Establecer nuestro correo y usuario de github en Git 3. Hcaer un git clone a nuestro repositorio de Github

  15. Cómo puedo crear un alias en git?

    git config --global alias.tree "log --graph --decorate --all --oneline"
